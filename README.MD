# Использование

## Создайте структуру папок:

input/ - поместите исходные фотографии

output/ - обработанные фотографии будут сохраняться здесь

Установите модули:

`npm i` или `yarn`

Запустите скрипт:

`node index.js`


## Особенности скрипта
Автоматическое определение центра: Изображения обрезаются из центра по минимальной стороне

Сохранение пропорций: <br>
После обрезки до квадрата, изображение масштабируется до 1024x1024

Поддержка вложенных папок: <br>
Сохраняет структуру папок из input в output

Оптимизация сжатия: <br>
Использует оптимальные настройки для разных форматов

Обработка ошибок: <br>
Продолжает работу даже при ошибках с отдельными файлами

Поддерживаемые форматы: <br>
JPG, PNG, WebP, TIFF, GIF
